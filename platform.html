Platform

JSX

JavaScript Syntax Extension is an extension of JavaScript that allows developers to write HTML right within JavaScript.

CSS

Cascading Style Sheets allows us to style HTML documents, but in our case JSX also supports it and will be used in a similar fashion.

Bootstrap

A CSS framework which will allow us to quickly create a responsive and mobile-friendly interface quickly so the focus can be on implementation of the back-end.

JavaScript

Our main programming language of choice which is used on both the client-side and server-side to make web pages interactive.

Node.js

A server side runtime environment based on JavaScript. It is used to utilize server-side RESTful APIs.

Express.js

A Node.js framework used to create secure, modular and fast applications with its routing simplification. It will be used to import libraries to implement logins, sessions and authentication.

React

A JavaScript library for building user interfaces. It is especially good for creating single-page applications. React is component-based and supports server-side.

MySQL

A relational database management system based on SQL. It will be our language of choice when querying the database.

Overleaf

Online LaTex editor. Used for writing, editing, and publishing scientific documents with collaboration and version control features.

Github

Where our source code will be hosted on. Includes collaborative and version control features.

Visual Studio Code

A code editor with useful features such as intelligent code completion and supporting git.
